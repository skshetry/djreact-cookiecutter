# Sambad

Setup



## How to setup


- Create a virtual env for this project
- Clone it
- `$ pip install -r requirements.txt`
- `$ cd ponynote/frontend`
- `$ npm install`
- `$ npm run start`
- `$ python manage.py runserver`


### Production use
 - `npm run build`
 - `python manage.py runserver --settings=sambad.settings.prod`
